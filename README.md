# LuxLodge (React + Vite)

This repository is the frontend for a hotel booking demo built with **React** and **Vite**. It includes a production build setup, ESLint rules, and Tailwind CSS for styling.

For detailed documentation, installation steps, and file-by-file explanations, see: [DOCUMENTATION.md](./DOCUMENTATION.md)

---

## Quick notes

- Start the dev server: `npm run dev`
- Build for production: `npm run build`
- Deploy: `npm run deploy` (uses `gh-pages`, make sure `gh-pages` is installed)
